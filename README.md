# Netflix Data Analysis

This project analyzes Netflix movies and TV shows data using Python.

## Objectives:
- Compare Movies vs TV Shows
- Identify top genres
- Analyze content growth over time

## Tools Used:
- Python
- Pandas
- Matplotlib

## Key Insights:
- Movies are more than TV Shows
- Drama and Comedy are most common genres
- Content increased rapidly after 2015

## Conclusion:
This project helps understand Netflix content trends using data analysis.
